# protocol

Auto-generated Python bindings for the SteelEagle protocol.

Contains all pb2 and pb2_grpc files for low-level interaction
with SteelEagle services. Messages represent datatypes, services
represent services either on the vehicle or on the server, and
testing contains datatypes used for test benches.
