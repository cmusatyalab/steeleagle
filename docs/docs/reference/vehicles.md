# Vehicles Supported

The following is a list of vehicles that have been (or plan to be) tested with SteelEagle.

## Legend
![Flight Tested](https://img.shields.io/badge/Flight%20Tested-ffffff?style=for-the-badge&color=00b100) Field tested in multiple real-world environments.

![Simulation Tested](https://img.shields.io/badge/Simulation%20Tested-ffffff?style=for-the-badge&color=f5e800) Tested using simulation tools.

![Theoretical Support](https://img.shields.io/badge/Theoretical%20Support-ffffff?style=for-the-badge&color=f57000) Untested, but theoretical support for this platform based upon its use of a simluation/flight tested autopilot that it shares.

![Roadmap](https://img.shields.io/badge/Roadmap-ffffff?style=for-the-badge&color=ff0000) Support for this platform is on our roadmap.

![Blue UAS](https://img.shields.io/badge/Blue%20UAS-ffffff?style=for-the-badge&color=005db1) This platform is Blue UAS compliant.

![Linux Client](https://img.shields.io/badge/Linux%20Client-ffffff?style=for-the-badge&color=812d45) Supports Linux-based offboard vehicle modules.

![Android Client](https://img.shields.io/badge/Android%20Client-ffffff?style=for-the-badge&color=287f85) Supports Android-based offboard vehicle modules.

![Onboard Client](https://img.shields.io/badge/Onboard%20Client-ffffff?style=for-the-badge&color=999999) Supports deploying the vehicle modules on the aircraft itself.
